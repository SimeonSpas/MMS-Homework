gcc -E main.c -o main.i ---> preproccessing
gcc -S main.c -o main.o ---> object code
gcc -C main.c -o main.s ---> assembly code

Sled tezi 3 stupki kodut preminava kum "Linkvane" koeto zarejda statichni ili dinamichni biblioteki, kato statichnite bilbioteki se zarejdat otdelno za vseki fail, a dinamichnite se zarejdat vednuj za vseki fail.
